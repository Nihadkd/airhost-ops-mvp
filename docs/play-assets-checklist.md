# Play Store Assets Checklist (ServNest)

Sist oppdatert: 23. februar 2026

1. App icon (512 x 512 PNG)
2. Feature graphic (1024 x 500 PNG)
3. Minst 2 mobil-screenshots (anbefalt 6-8)
4. Kort beskrivelse
5. Full beskrivelse

1. Dashboard med filter
2. Oppdragsdetaljer
3. Chat mellom partene
4. Fullførte oppdrag
5. Profil/innstillinger
6. Språkbytte NO/EN
