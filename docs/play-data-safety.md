# Play Console - Data Safety (ServNest)

Sist oppdatert: 23. februar 2026

## Formål
Dette er et praktisk utkast for Data safety-skjema i Google Play Console. Gå gjennom og juster ved endringer i appfunksjon.

## Data som behandles i ServNest
1. Personlig info
- Navn
- E-post
- Telefonnummer

2. Meldinger
- Oppdragschat mellom utleier og tjenesteutfører

3. Bilder / filer
- Bilder lastet opp på oppdrag
- Kommentarer knyttet til bilder

4. App-aktivitet
- Oppdrag opprettet, tildelt, fullført
- Varsler i appen

5. Enhet / app-identifikatorer
- Push-token for varsler

## Bruksformål (typisk avkrysning i Play)
1. App-funksjonalitet
- Brukes for konto, oppdrag, chat, bilder og varsler.

2. Kundestøtte
- Kontaktinformasjon brukes ved supporthenvendelser.

3. Sikkerhet og svindelforebygging
- Data brukes for tilgangskontroll og sporbarhet.

## Deling av data
Per nå: Ikke delt med tredjeparter for annonsering.
Tekniske underleverandører (hosting/push) brukes kun for drift av tjenesten.

## Datakryptering
Data sendes over HTTPS mellom klient og backend.

## Sletting av data
Bruker kan be om sletting via support.
Supportkontakt: support@servnest.app

## Viktig før innsending
1. Sørg for at svar i Play Console matcher faktisk kode og backend-rutiner.
2. Oppdater dette dokumentet ved nye datatyper eller nye integrasjoner.
