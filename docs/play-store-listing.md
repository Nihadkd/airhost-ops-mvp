# ServNest - Play Store Kladd

## App Name
ServNest

## Short Description (maks 80 tegn)
Administrer oppdrag, chat og varsler mellom utleier og tjenesteutfører.

## Full Description (NO)
ServNest gjør det enkelt å håndtere oppdrag på tvers av utleiere, tjenesteutførere og admin.

Med ServNest kan du:
- Opprette, redigere og følge opp oppdrag
- Tildele oppdrag til riktig tjenesteutfører
- Bruke innebygd chat for hvert oppdrag
- Motta varsler ved nye meldinger og statusendringer
- Laste opp bilder og dokumentere utført arbeid
- Filtrere og søke i oppdrag

ServNest er bygget for tydelig kommunikasjon, rask oppfølging og bedre kvalitet i driften.

## Full Description (EN)
ServNest helps teams manage jobs between landlords, service workers, and admins.

With ServNest you can:
- Create, edit, and track jobs
- Assign jobs to the right worker
- Use built-in job chat
- Receive notifications for messages and status updates
- Upload photos for job documentation
- Filter and search jobs quickly

ServNest is designed for clear communication, fast follow-up, and better operational quality.
